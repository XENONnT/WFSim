=================================
Welcome to wfsim's documentation!
=================================

Github page: https://github.com/XENONnT/WFSim

**WFSim** is the wavefrom simulation framework for XENONnT, built on top of the `straxen framework <https://github.com/XENONnT/straxen>`_.

Installation
==================

1. To get started for simulating XENONnT please follow the `straxen setup guide <https://straxen.readthedocs.io/en/latest/setup.html>`_. Simulating XENON1T works out of the box.

2. On top of ``straxen``, install ``wfsim``:

.. code-block:: bash

   pip install wfsim
